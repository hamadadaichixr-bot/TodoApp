# デスクトップ型タスク管理（To-Do）アプリケーション

Java Swing（GUI）を用いて開発した、デスクトップ型のタスク管理アプリです。

## 1. 機能概要
* **タスクの追加・削除:** 入力欄から文字を入力してタスクを追加し、個別の削除ボタンで削除できます。
* **データの永続化（自動保存・読み込み）:** アプリを終了しても、次回起動時に自動で前回のタスクリスト（`todo.txt`）を読み込みます。
* **スクロール機能:** タスクが大量に増えても、画面が崩れずスクロールで確認できます。

## 2. 技術スタック
* **言語:** Java (Java SE)
* **GUIフレームワーク:** Java Swing / AWT
* **開発ツール:** Eclipse / SourceTree / Git

## 3. 工夫・こだわった点
* **例外・エラー回避:** 空文字が追加されないようにバリデーション処理を実装。
* **リソース管理:** ファイル入出力の不具合（メモリリーク）を防ぐため、`try-with-resources`文を採用して安全にファイルをクローズする設計にしました。
* **コードの共通化:** 起動時の読み込みとボタン押下時の追加処理を共通メソッド化し、コードの保守性を高めました。
